# home cil

An Android application built with Kotlin using XML Layouts.

## Requirements
- Android Studio Hedgehog (2023.1.1) or later
- JDK 17 or later
- Android SDK API 36
- Gradle 8.14.4

## Getting Started
1. Open the project in Android Studio
2. Sync Gradle files
3. Build and run with Android Studio or `./gradlew assembleDebug`
